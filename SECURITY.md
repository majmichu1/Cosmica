# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅ |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a private discussion or emailing the maintainer.

**Do NOT open a public issue** for security vulnerabilities.

We will respond within 48 hours and work with you to resolve the issue.

## Security Considerations

### Model Downloads
Cosmica can download AI model weights. We verify downloads via SHA-256 checksums.

### License System
Cosmica has NO invasive DRM. License status is cached locally and trusted when the validation server is unavailable.

### Data Privacy
- All processing happens **locally** on your machine
- No images are uploaded to any server
- AI model training uses only your local FITS files
- No telemetry or analytics are collected
